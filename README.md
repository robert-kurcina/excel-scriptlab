# MEST Tactics :: Excel ScriptLab Parser

This is a ViteJS + ReactJS Web application. 

```
cd ~/projects
git clone https://github.com/robert-kurcina/excel-scriptlab
cd excel-scriptlab
npm install
npm run dev
```

Open the browser to `localhost:5172` to play with the functionality.
